**What it does:** Web-grounded AI search API that returns real-time information with citations. Unlike standard LLM APIs, Sonar searches the live web before answering, making it ideal for market news, earnings data, and sentiment analysis.

**Integration difficulty:** LOW. Simple REST API, similar to any LLM API call. Drop-in replacement for situations where you need real-time data instead of static model knowledge.

**Monthly cost:**
- Sonar: $1 input / $1 output per million tokens
- Sonar Pro: $3 input / $15 output per million tokens
- Search API: $5 per 1,000 requests
- Realistic monthly cost for EdgeFinder: $10-30/month depending on query volume

**What it unlocks for waitlist onboarding:** Real-time market sentiment and news summaries for any ticker. A waitlist user asks "What's the latest on AAPL?" and EdgeFinder returns a sourced, current answer instead of stale training data. This is a differentiator most competitors do not have.
